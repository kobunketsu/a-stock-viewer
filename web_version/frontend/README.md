## 计划
- 使用 Vite + React + TypeScript + Ant Design 构建页面。
- 引入 ECharts、React Query、Zustand/Redux Toolkit、KeyboardJS 等依赖。
- 页面：自选列表、日 K、分时，支持多标签与快捷键。
- 响应式：使用 CSS Grid/Flex，适配桌面与移动端；触控交互启用。
